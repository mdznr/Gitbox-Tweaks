## Gitbox Tweaks

SIMBL plugin for Gitbox. Improves support for Retina displays and Yosemite.

### Installation

1. Install [EasySIMBL][].
2. Download [Gitbox Tweaks][Tweaks].
3. Open EasySIMBL.
4. Drag and drop plugin onto EasySIMBL.
